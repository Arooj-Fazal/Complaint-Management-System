
# 🏛️ University Complaint Management System

A modern, web-based platform designed to streamline communication between students, teachers, and university administration. This system allows users to submit, track, and manage complaints efficiently.

## 🔗 Live Demo
Check out the live project here: [https://arooj-fazal.github.io/Complaint-Management-System/](https://arooj-fazal.github.io/Complaint-Management-System/)

## ✨ Features
* **Multi-Role Dashboards**: Specific interfaces for Students, Teachers, HODs, and Deans.
* **Real-time Interaction**: Built with Firebase to ensure data is updated instantly.
* **Modern UI/UX**: Clean and professional design using HTML and CSS.
* **Secure Access**: Integrated login and registration system for different roles.

## 🛠️ Built With
* **Frontend**: HTML5, CSS3, JavaScript
* **Backend**: Firebase (Authentication & Firestore)
* **Version Control**: GitHub & GitHub Desktop

## 📂 Project Structure
* `index.html` - The main landing page.
* `adminDashboard.html` - Management portal for administrators.
* `deanDashboard.html` - Oversight dashboard for Deans.
* `img/` - Contains all project images and icons.

## 🚀 How to Run Locally
1. Clone this repository to your local machine.
2. Open the folder in **VS Code**.
3. Use the **Live Server** extension to launch the `index.html` file.
